This project is a Machine Learning based web application that predicts the estimated monthly rent of a house based on various parameters such as BHK type, location, furnishing status, tenant preference, and number of bathrooms. The system helps users get an approximate rent value and recommendations for available rental houses.

The project was developed as part of a hackathon to demonstrate how Machine Learning models can be integrated with a web application to solve real-world problems in the housing rental market.

The dataset used for training the model was collected from the platform Kaggle. The model was trained using supervised learning techniques and deployed using a lightweight web framework.

The application provides a simple and user-friendly interface where users can enter their preferences and receive rent predictions along with suggested rental options.

Technologies Used:
Python – For machine learning and backend logic
Machine Learning (Linear Regression) – For rent prediction
Flask – To build the web application
HTML – To structure the web pages
CSS – To design and style the interface
Kaggle Dataset – For training the prediction model

Features of the Project:
User login interface
BHK selection
Location selection
User details input
Rental house recommendations
Estimated monthly rent prediction
Simple and responsive user interface
